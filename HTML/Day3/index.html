Inline CSS

There are 3 types of CSS

Inline CSS
Inline using on element directly with "style" attribute. Note: Having a higher priority
Internal CSS
Internal CSS is using with <style></style>. Note: Having a higher priority over External CSS
Extenal CSS
External CSS is using with file like index.html or script.js but it have style.css .css is extension and its mandatory. Note: It is best practice to use it.


Semantic Tags

Semantic elements = elements with a meaning.

What are Semantic Elements?

A semantic element clearly describes its meaning to both the browser and the developer.
Examples of non-semantic elements: div and span - Tells nothing about its content. Examples of semantic
elements: img, table, and article - Clearly defines its content.

Semantic tags having:

  • header
  • nav
  • footer
  • main
  • article
  • details
