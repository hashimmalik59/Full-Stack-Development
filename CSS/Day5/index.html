Flex Wrap and Flex Flow

Flex Flow is the shorthand of Flex wrap and Flex flow



Justify Content:

Horizontal Alignment, Row, x-axis, Cross-axis



Align Items:

Vertical Alignment, Column, yx-axis, Main-axis



Align Content:

Align Conten will be worked when wrap is on it will apply between row/column

Note: It always apply with flex wrap



Align Self:

Vertical alignment for individual box(child).



Flexbox Order

It will change the order of individual box(child) like: lower to higher.



Flex Grow:

It increase the size(width) of an individual box(child). eg: if we give value "2" then it will be grow 2x than others.



Flex Basis:

In CSS Flexbox, flex-basis defines the initial main size of a flex item before any space is distributed according to flex-grow or flex-shrink.



Flex Shrink:

It decrease the size(width) of an individual box(child). eg: if we give value "2" then it will be shrink 2x than others.
