CSS Formatting

CSS formatting—how to properly write and structure CSS code:

selector {
property: value;
}

✅ Formatting Best Practices:
Use consistent indentation (2 or 4 spaces is common)
Add a space after colons:

color: red; /* not color:red; */

background-color: black; /* not Background-Color */

Separate sections with comments (optional but helpful):

/* Typography */
h1, h2, h3 {
font-weight: bold;
}
