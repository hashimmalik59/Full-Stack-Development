CSS Box Model

Margin:
Space between border and other boxes outside of it.
Creates distance between elements.
Border:
The line wrapping padding and content.
Padding:
Distance between border and content.
Increases the size of box.
Background color applies to the padding area.
Content:
The actual content of the box(text, images, etc).
Controlled with properties like width and heigth.
